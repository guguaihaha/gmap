创建地图
e.g:

var elem = document.getElementById("warp");

var mapname = new gmap({

target:elem,

center:"45,45",

zoom:7

});

参数

target

Type: HTMLelement

创建地图目标. Required.
center

Type: string

初创地图的中心点坐标,例如:"39,2934849,20,238388"
zoom

Type: number

初创地图的缩略视角数值是0~15
draggable

Type: boolean

是否允许地图被拖拽操作
draggableCursor

Type: string

可以是个url地址,例如:draggableCursor: 'url(http://www.example.com/icon.png), auto;'
keyboardShortcuts

Type: boolean

是否允许键盘操作地图,默认是开启状态
mapMaker

Type: boolean

启用后将替换传统的纹理
mapTypeId

Type: MapTypeId

地图的展现方式:1:d:google.maps.MapTypeId.ROADMAP(默认),
2:d:google.maps.MapTypeId.HYBRID,
3:d:google.maps.MapTypeId.SATELLITE,
4:d:google.maps.MapTypeId.TERRAIN.
maxZoom

Type: number

最大视角数值范围"0~15"或者为"null"
minZoom

Type: number

最大视角数值范围"0~15"或者为"null"
mapMaker

Type: boolean

启用后将替换传统的纹理
panControl

Type: boolean

开关控制
rotateControl

Type: boolean

旋转控制
scaleControl

Type: boolean

缩放控制
scrollwheel

Type: boolean

是否启用滚轮缩放,默认启用
streetViewControl

Type: boolean

是否显示街道
styles

Type: Array

自定义地图样式,详情参见:https://developers.google.com/maps/documentation/javascript/3.exp/reference#MapTypeStyle
mapMaker

Type: boolean

启用后将替换传统的纹理
tilt

Type: number

地图倾斜视角0~45
zoomControl

Type: boolean

是否开启缩放控制
