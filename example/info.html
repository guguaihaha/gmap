Properties

content

type:html Elements

required,将要展示的节点内容
maxWidth

type:number

显示框的最大宽度
pixelOffset

type:number

position

type:number

显示框的位置
zIndex

type:number

显示框的叠加层数
marker

type:object

将在哪个marker对象出现显示层
clickShow

type:boolean

是否调用事件去显示浮层
clickEvent

type:string

事件名称"click"
success

type:

当每个mark创建完info会触发一次改行为

Method

getInfo

return Array;

返回地图上当前info数组对象

removeInfo(infoObject)

移除对应的info对象;mapname.removeInfo(infoObject);如果infoObject为空则移除当前所有info对象
