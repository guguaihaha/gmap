Properties

html

type:html Elements

required,将要展示的节点内容
markArray

type:[Array]

针对哪些节点做这些修改,如果为空则使用地图全部mark对象
fn

type:function

当每个mark创建完info会触发一次改行为

Method

setInfoBubble(properties)

Type: object

return info;

此方法直接调用mapname.setInfoBubble(properties)即可
