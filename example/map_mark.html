创建所有坐标
e.g:

var elem = document.getElementById("warp");

var mapname = new gmap({

target:elem,

center:"45,45",

zoom:7

});

mapname.onMapLoad = function(){

mapname.setMarker([{

position:"45,45"

}])

}

--------------------

mapname.setMarker(arrayProperties)

    arrayProperties:[Properties,Properties]

Properties

position

Type: String [LatLng]

中心点,此选项必填;e.g:"23.34344,22,494948"

icon

Type: object

此对象包含4个对象

url:icon的url地址;

size:new google.maps.Size(20, 32);

origin:new google.maps.Point(0, 0);

anchor:new google.maps.Point(0, 32);

shape

Type: MarkerShape

图像映射区域定义用于拖动/点击。

title

Type: string

标题

zIndex

Type: number

所有标记在地图上显示的顺序zIndex,较高的值显示在前面的标记值较低。默认情况下,根据其垂直位置标记显示在屏幕上,较低的标志出现在面前的标记在屏幕上。

draggable

Type: boolean

是否允许标点被拖拽,数值:"true,false";默认是true

clickable

Type: boolean

是否允许标点被点击,数值:"true,false";默认是true

cursor

Type: string

鼠标悬停的样式,可以自定义url或者"auto".

Method

removeMarker

Type: Array[Object]

return none;

移除传入的marker对象,如果marker对象没有传入参数,则删除地图上所有坐标点

getMarker

返回地图上marker数组对象
